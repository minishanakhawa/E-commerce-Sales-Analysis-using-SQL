E-commerce Sales Analysis (SQL Project)

Description
This project analyzes e-commerce data using SQL to understand customer behavior and sales trends.

 Tools Used
- MySQL

Key Analysis
- Top customers by spending
- Total revenue calculation
- Orders per city
- Monthly revenue trends
- Most sold products
- Customers with no orders

SQL Concepts Used
- JOIN (INNER, LEFT)
- GROUP BY
- Aggregate Functions (SUM, COUNT, AVG)
- ORDER BY
- Subqueries

Outcome
Derived meaningful insights about customer purchases and product performance.
